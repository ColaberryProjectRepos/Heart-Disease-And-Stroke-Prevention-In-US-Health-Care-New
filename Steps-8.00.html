1. Staging layer - This is where the raw data is gathered from one or more sources and combined into a database with
little to no transformation.

2. ODS layer - ODS stands for Operational Data Store. This layer consist of real time data, with real time defined by the
the requirements of the organization. Here we will transform and validate the data by using ETL transformations
and applying business logic.

3. Data Warehouse - This is the where the data enters into the data warehouse which holds historical data received
from the ODS layer used for analysis over larger time frames.

 

The Applications used to create the project are:

1. SSMS - used to run SQL scripts

2. SSIS - an ETL tool used to transform and validate data as well as load SQL tables.
