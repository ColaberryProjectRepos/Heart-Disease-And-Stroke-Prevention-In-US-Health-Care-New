Configure Staging ETL Data Flow Task 12




Step 1: Configure the Staging Layer Data Flow Task.

A. Double Click the Data Flow Task named Data flow task_JSON.

B. Drag and drop a Script Component and name it Script Component - JSON Extraction. Specify the use as Source.

C. Drag and drop an OLEDB Destination and name it OLE DB Destination_Tbl_Stg_HeartDisease_Stroke

D. Connect them as shown in the image below.



Step 2: Configure the Script Component.

A. Double Click on the Script Component name Script Component - JSON Extraction.

B. Click in the ReadOnlyVariables section and select User::Full_FilePath.

C. Click the Inputs and Outputs Tab.

D. Add all Output Columns and set the Data Types and Lengths as string [DT_STR] and 255 respectively.


E. Click Edit Script.



Step 3: Add Code to main.cs

A. Delete all the code in main.cs.

B. Copy and Paste the code, from the Main Script task load Stg_tbl_Heart_Stroke file in the Staging Folder, into main.cs.


Step 4: Add necessary Extensions.

With the main.cs still open.

A. Under the Solution Explorer right click on Add References.

B. Select System.Web.Extensions and click OK.

Step 5: Add a Class.

With the Main Script still open

A. Under the Project tab select Add Class.

B. Select Class, name it HeartDisease.cs and click Add.


Step 6: Add Code to the Class

A. Delete the default code in Class and then Copy and Paste the code, from the class script - JSON file in the Staging
Folder, into HeartDisease.cs.

B. Press the Save All button.

C. Close the window.

D. Click the OK button in the Script Transformation Editor.


Step 7: Configure the OLE DB Destination named OLE DB Destination_Tbl_Stg_HeartDisease_Stroke

A. Double Click on an OLE DB Destination.

B. Set the OLE DB connection manager to HEART_SROKE_STG

C. Set the Name of the table to [dbo].[ Tbl_Stg_HeartDisease_Stroke].



Step 8: Configure the Mappings for the OLE DB Destination named OLE DB Destination_Tbl_Stg_HeartDisease_Stroke.

A. Click the Mappings tabs.

B. Set the Mappings as shown below.

C. Click OK.
